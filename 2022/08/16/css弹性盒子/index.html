css弹性盒子

css弹性盒子

  • 弹性盒子可以解决那些问题？
  1. 使盒子中的元素水平垂直居中对齐
  2. 使盒子中元素之间的空隙平均分配
  3. 使元素自动占据盒子剩余空间
  • 定义弹性盒子
属性 描述 设置参数
display 将普通盒子转化为弹性盒子 flex
flex-direction 设置行布局, 列布局 row, row-reverse，column, column-reverse
flex-wrap 设置弹性元素是否折叠 nowrap, wrap, wrap-reverse
flex-flow 同时设置flex-direction, flex-wrap两个属性的值 eg: row wrap
justify-content 元素在主轴的对齐方式 flex-start(主轴左边), center(中心位置), flex-end(主轴右边), space-between, space-around
align-items 元素在辐轴上的对齐方式 flex-start, center, flex-end, stretch(不需要设置元素高度，会自动铺满容器的那一列)
align-content 设置多行元素在容器中的整齐对齐方式 flex-start, center, flex-end, stretch, space-between, space-around
  • flex-direction 属性布局


  • align-items 属性布局

弹性元素

  1. flex-grow 属性
  • 元素被拉大的比例，按比例分配容器剩余空间
    • 默认值为零元素不占用空间
    • 默认值为 n 占用空间中 n 份空间

下图是上面css代码中三个元素的容器空间占比

  1. flex-shrink
  • 元素被压缩的比例
    • 默认为 1 等比压缩
    • 设置为 0 表示不压缩
  1. flex-basis
  • 元素在主轴上的默认尺寸，其优先级高于 width 属性
  1. flex
  • flex: flex-grow flex-shrink flex-basis
  1. order
  • 子元素在容器中的排列顺序，order值越小越靠前。
  1. align-self
  • 设置单个弹性元素在辅轴上的对齐方式， align-itmes 设置的是全部弹性元素
